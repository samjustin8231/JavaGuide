# redis如何删除过期数据
缓存会有淘汰算法，数据一般不会永远存储在内存中，内存资源是宝贵的。

定期删除+惰性删除

* 定期删除：redis默认每个100ms，随机抽取一些设置了过期时间的key，如果过期了就删除；(那么有些key就不会抽到)
* 惰性删除：获取某个key的时候，会检查是否过期，如果过期了，则删除(并不是到期了直接自动删除，而是获取的时候惰性删除)
